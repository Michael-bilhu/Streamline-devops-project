Cloud Deployment

Hosted on an AWS EC2 Ubuntu server and exposed publicly through HTTP traffic on port 80.

Dockerized Application

The frontend is packaged using a Dockerfile and served through an Nginx container.

Backend API

A Node.js Express backend provides API responses and runs as a separate container.

Nginx Reverse Proxy

Nginx serves the frontend and routes API requests to the backend service internally.

Docker Compose Stack

Docker Compose manages multiple services: frontend, backend API, and Redis.

CI/CD Automation

Jenkins detects GitHub changes, connects to EC2 through SSH, and redeploys the app automatically.
